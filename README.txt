ORION LIVESTOCK V7.1 OFFICIAL
BẢN ĐÓNG BĂNG V7 – DÙNG ĐỂ CHIA SẺ QUA ZALO

CÁCH DÙNG TRÊN MÁY TÍNH
1. Giải nén toàn bộ file ZIP.
2. Mở thư mục ORION_LIVESTOCK_V7_1_OFFICIAL.
3. Nhấp đúp file index.html.
4. Chọn mở bằng Google Chrome hoặc Microsoft Edge.

CÁCH DÙNG TRÊN ĐIỆN THOẠI ANDROID
1. Tải file ZIP về máy.
2. Giải nén bằng ứng dụng Tệp/My Files.
3. Mở file index.html bằng Google Chrome.
4. Có thể chọn Thêm vào màn hình chính để mở nhanh.

LƯU Ý QUAN TRỌNG
- Dữ liệu được lưu riêng trên từng thiết bị.
- Gửi file cho người khác không làm đồng bộ dữ liệu giữa hai máy.
- Không xóa dữ liệu trình duyệt nếu chưa sao lưu.
- Nên dùng nút Sao lưu dữ liệu định kỳ.
- Khi xuất PDF, chọn khổ A4 ngang và bật đồ họa nền nếu trình duyệt có tùy chọn này.

TÍNH NĂNG
- 6 nhà nuôi.
- Theo dõi đàn heo, cám, kho cám tổng, sức khỏe và ghi chú.
- Cài đặt ngày nhập, tuần tuổi nhập và mã đàn tự động.
- Xuất PDF 2 trang A4 ngang.
- Xuất Excel.
- Sao lưu và khôi phục dữ liệu.
- Đã sửa lỗi nhập số liên tục như 500, 1000, 1500.
